1. Objective

The purpose of this assignment is to apply automation testing concepts using Playwright to validate the functionality of web applications. Key objectives include:

Automating repetitive UI testing tasks

Ensuring cross-browser compatibility

Generating accurate and reproducible test results

2. Tools and Technologies Used

Playwright – Browser automation and testing

Node.js – JavaScript runtime environment

Visual Studio Code – IDE for development

Git & GitHub – Version control and assignment submission

Browsers Tested: Chrome, Firefox, Edge

3. Project Overview

This project contains scripts that automate common user actions on web applications, such as:

Navigating pages

Filling and submitting forms

Validating messages and UI elements

Capturing screenshots for verification

The scripts ensure reliability and consistency in web application testing, reducing manual testing effort.

4. Key Features / Test Scenarios

Page Navigation – Open web pages and validate titles and headers

Form Automation – Automatically fill input fields and submit forms

Validation Checks – Confirm text, buttons, and UI components appear correctly

Screenshot Capture – Capture screenshots of test results for reporting

Cross-Browser Testing – Execute tests in Chrome, Firefox, and Edge

5. Setup and Execution Steps

Clone the Repository:

git clone https://github.com/hasinidilrukshi/playwright-assignment.git


Navigate to the Project Directory:

cd "IT23281578_playwright code_3.01_WE"
